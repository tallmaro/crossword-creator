# crossword-creator

A [Claude Code](https://claude.com/claude-code) skill that builds **real, solvable crossword puzzles** on any theme — a symmetric grid where every Across *and* Down answer is a genuine word, plus hand-written clues, delivered as structured JSON and an interactive HTML player you can actually solve in a browser.

The hard part of a crossword is the fill: every white square has to belong to a real word in *both* directions, with proper rotational symmetry. This skill does that with a backtracking constructor over a **scored** crossword wordlist (567k entries), keeping only high-quality fill and selecting the cleanest of many candidate grids. Claude writes the clues — the part that actually wants human judgment.

> Ask Claude *"make me a crossword about jazz"* and it runs the whole pipeline: picks theme answers, builds a valid grid, writes the clues, and hands you `crossword.json` + a playable `index.html`.

## What you get

- **`crossword.json`** — structured puzzle data (grid, numbering, clues with positions and answers). Drop it into any UI. See [`schema`](skills/crossword-creator/references/schema.md).
- **`index.html`** — a self-contained player: type to fill, click clues to navigate, Check / Reveal / Clear, autosaves progress. No build step, no dependencies.

## Features

- **Any theme**, themed or themeless.
- **Sizes**: 15×15 (standard), 11×11 (midi), 7×7 (mini).
- **Difficulty**: affects how Claude clues (Monday-easy through Thursday-tricky).
- **Guaranteed-valid grids** — 180° rotational symmetry, every entry a real word, verified before clues are written.
- **Quality-selected fill** — pulls only from entries the wordlist rates "genuinely good," and keeps the cleanest grid out of many.

## Install

**Option A — as a plugin (recommended):**

```
/plugin marketplace add tallmaro/crossword-creator
/plugin install crossword-creator@crossword-creator
```

**Option B — manual:** clone and drop the skill into your personal skills folder.

```bash
git clone https://github.com/tallmaro/crossword-creator.git
cp -r crossword-creator/skills/crossword-creator ~/.claude/skills/
```

(For a single project, `.claude/skills/` at the repo root works too.)

## Usage

Once installed, just describe what you want:

> "Create a 15×15 crossword themed around Greek mythology."
> "Make me a quick mini puzzle about coffee."
> "Build a Wednesday-difficulty crossword about jazz musicians."

Under the hood the skill runs two scripts (you don't have to call them by hand, but you can):

```bash
# 1. construct a valid grid with your theme answers locked in
python skills/crossword-creator/scripts/build_grid.py \
    --theme "ALFREDHITCHCOCK,STEVENSPIELBERG" --size 15 --out ./my-puzzle

# 2. (write a clue per entry into my-puzzle/clues.json, then)
python skills/crossword-creator/scripts/assemble.py --dir ./my-puzzle \
    --title "Lights, Camera, Crossword" --difficulty monday \
    --theme "Legendary film directors"
```

Open the result by serving the folder over HTTP:

```bash
python3 -m http.server 8001 --directory ./my-puzzle   # then visit http://localhost:8001
```

## How it works

1. **Theme placement** — your theme answers are placed as Across entries in symmetric rows (grid-spanning answers, or equal-length mirror pairs) and locked.
2. **Skeleton** — a symmetric block pattern is generated by splitting over-long runs.
3. **Fill** — backtracking with most-constrained-slot-first ordering, one-step feasibility pruning, and common-words-first selection, over a scored wordlist. A short per-grid timeout means hard skeletons are abandoned fast and easy ones kept.
4. **Quality selection** — many grids are generated; the one with the fewest low-score entries wins.
5. **Clues** — Claude writes one clue per entry, matched to the chosen difficulty.
